For this project, I was required to make an infographic on some sort of communication technology. I chose the TV remote. As shown in the infographic below, the TV remote function begins when a user presses a button on the remote. A chip then detects the button being pressed and translates it into code. Then the transistor amplifies the signal of the code. After this, the LED sends the code to the TV through infrared light. Finally, the TV recieves the code and completes the request.

I created this infographic using Adobe InDesign. InDesign was the best software to create the layout I needed for this infographic. Learn more about InDesign here. I also used Adobe Photoshop in order to clip backrounds out of photos used in the infographic when necessary.
